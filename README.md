# Expense Tracker (MERN)

> Full stack expense tracker

## Usage

```
change config.env file in config folder
```

```
 npm install
 cd front npm install
 cd ..
 
 # Run front and backend
 npm run dev
 
 # Backend only
 npm run back
 
 # Frontend only
 npm run front
 
 # Build front
 cd front
 npm run build
 
 # Prod
 npm start
```
